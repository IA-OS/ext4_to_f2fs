EXT to F2FS ROM converter
=============
This is a simple tool for converting EXT4 roms to F2FS.

Compatability
-----
* Paranoid Android (Since 4.5-BETA1)

Usage
-----
Linux:

    cd linux
    chmod +x convert.sh
    ./convert.sh [ROM_NAME].zip

Windows:

    Run command prompt as administrator
    cd windows
    convert.bat [ROM_NAME].zip
